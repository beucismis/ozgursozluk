About

This project is free alternative Ekşi Sözlük front-end focused on privacy and performance. The source is available on GitHub at https://github.com/beucismis/ozgursozluk

Made with <3 on the T61.

  • No JavaScript
  • Docker support
  • Topic searching
  • Viewing topic, entry and author
  • Gündem and Debe page support
  • Optional displaying author nickname and more
  • 8 different theme support
  • Self-hosted, ad-free, simple and fast
  • Responsive support for small screens

This project does not have an official affiliation with eksisozluk.com.
